# Naming Rules

## Canonical Object Names
Use `Case`, `Signal`, `EvidencePack`, `RecommendationDraft`, `ApprovalRecord`, and `AuditEvent` across docs, code, and demos.

## File and Identifier Rules
- Use kebab-case for file names.
- Use stable IDs such as `REQ-001`, `F-001`, `GC-001`, and `RISK-PRIVACY-001`.
- Do not rename approval or audit concepts in judge-facing material.
