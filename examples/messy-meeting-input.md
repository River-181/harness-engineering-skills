# Messy Meeting Input Example

We need something for relationship managers. They see a lot of small-business borrowers and risk signals, but the data is scattered. Maybe the system should watch for policy news, repayment risk, fraud signals, and draft a customer message. But we cannot let AI send anything directly. The judges want AI Agent behavior, so it needs to make a judgment and take an action, then verify. It should be demoable in five minutes.
