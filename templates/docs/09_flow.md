## Core Demo Flow

1. Input arrives.
2. System interprets context.
3. Evidence is retrieved.
4. Agent makes a judgment.
5. Agent selects or drafts an action.
6. Validator checks quality, policy, and evidence.
7. Human approves, rejects, or requests more evidence.
8. System logs the decision.
9. Case or object state updates.
10. Eval feedback is captured.
