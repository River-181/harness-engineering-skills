# Common Completion Report

Every skill run must end with a short completion report.

```markdown
| Area | Result |
|---|---|
| Files created/updated | |
| Decisions made | |
| Assumptions | |
| Validation performed | |
| Remaining risks | |
| Recommended next skill | |
```

Keep the report factual. Do not claim that a file was inspected, created, updated, or validated unless that happened in the current run.
