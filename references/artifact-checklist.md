# Artifact Checklist

## Required lifecycle artifacts

- `docs/00_source-log.md`: raw source preservation and assumptions.
- `docs/01_meeting-log.md`: decisions, requirements, open questions, next actions.
- `docs/02_cps.md`: Context → Problem → Solution.
- `docs/03_principles.md`: non-negotiable project principles and enforcement.
- `docs/04_definitions.md`: shared vocabulary and anti-synonyms.
- `docs/05_domain-model.md`: entities, states, events, permissions.
- `docs/06_prd.md`: product requirements traced to CPS.
- `docs/07_architecture.md`: components, dataflow, boundaries, failure handling.
- `docs/08_feature-spec.md`: implementation-ready feature specs.
- `docs/09_flow.md`: core user and agent flows.
- `docs/10_eval-plan.md`: golden cases, metrics, rubrics, regression rules.
- `docs/11_change-log.md`: change date, target, content, reason, impact.
- `docs/12_handoff.md`: runbook, demo script, known limitations, next steps.
