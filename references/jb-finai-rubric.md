# JB Fin:AI Rubric Adapter

Use this only for JB Fin:AI-style financial AI Agent competition or presentation work.

## Scoring categories

Each core category is worth 20 points:

1. Topic fit and problem definition.
2. Financial business linkage and customer value.
3. AI Agent design and technical feasibility.
4. MVP completeness and verifiability.
5. Innovation, scalability, and operational risk management.

Demo video can add 5 points when applicable.

## Required AI Agent elements

Every submitted service should show:

- judgment: interpret input, situation, or data into a state, risk, priority, or decision;
- action: select the next action such as retrieval, question, draft, alert, summary, booking, or modification;
- verification/improvement: check result quality, retry when needed, or terminate safely.

## Required document shapes

MVP proposal:

1. Summary
2. Problem Definition
3. Solution Overview
4. Key Features
5. Data & Tech Plan
6. User Scenario
7. Expected Impact

Feature specification:

1. Service
2. Architecture
3. Feature Specification
4. Flow
5. Future Work
6. Appendix

## Hard risks

- Missing required deliverables.
- Demo cannot run on site.
- Feature changes not recorded in change history.
- Unauthorized copying of other teams' code, docs, or feature specs.
- Privacy, security, copyright, hallucination, explainability, or responsibility risk ignored.
