# JB Fin:AI Submission Checklist

- [ ] Problem, target user, and usage situation are concrete.
- [ ] JB financial business linkage is explicit.
- [ ] AI Agent judgment/action/verification is visible.
- [ ] Data, RAG, rules, models, APIs, licenses, and constraints are described.
- [ ] MVP proposal and feature spec are consistent.
- [ ] Demo can run on local equipment.
- [ ] Change history is complete.
- [ ] Privacy, security, copyright, hallucination, explainability, responsibility risks are addressed.
