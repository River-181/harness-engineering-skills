# Agent Loop Cards

## Retrieval agent

- Trigger: missing evidence.
- Judgment: which source is needed.
- Action: retrieve/search/request.
- Verification: source validity and recency.

## Drafting agent

- Trigger: approved recommendation path.
- Judgment: user intent and policy constraints.
- Action: draft message/action.
- Verification: tone, factuality, policy, approval state.

## Validator agent

- Trigger: any generated output before display or action.
- Judgment: pass/warn/fail.
- Action: approve for review, request revision, or block.
- Verification: rubric and evidence checks.
